###适配器

    Convert the interface of a class into another interface clients expect.
    Adapter lets classes work together that couldn't
    otherwise because of incompatible interfaces.

    将一个类的接口变换成客户端所期待的另一种接口，从而使原本因接口不匹配而无法
    在一起工作的两个类能够在一起工作。

    1. Target目标角色

    2. Adaptee源角色

    3. Adapter适配器角色
